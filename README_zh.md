# Stick Hero: Pressure Edition (棒子英雄：压感版)

[![English](https://img.shields.io/badge/English-blue)](README.md)　[![中文](https://img.shields.io/badge/中文-orange)](README_zh.md)

## 🎮 在线试玩

**[点击这里开始游戏](https://stick-hero-pressure.vercel.app/)**

一款基于压感触控的 Stick Hero 改编游戏，通过按压力度控制棍子长度。使用 HTML、CSS 和 JavaScript 构建。

![游戏演示](gameplay-demo.gif)

## 系统要求

⚠️ **本游戏需要：**
- 配备 Force Touch 触控板的 Mac
- Safari 浏览器

其他浏览器或设备不支持本游戏所使用的压感触控板 API。

## 玩法说明

1. **按住**触控板开始伸长棍子
2. **按压力度越大，棍子越长**——压力直接控制棍子长度
3. 倒计时会出现在屏幕上，当倒计时归**零**时，棍子长度锁定并倒下
4. 如果棍子能够到下一个平台，英雄就会走过去；否则游戏结束！
5. 落在平台中央的**红色区域**可获得**双倍积分**

## 小技巧

- 轻按 = 短棍子
- 重按 = 长棍子
- 注意倒计时——归零后无法再改变长度
- 瞄准中央红色区域以获得更高分数
